0.0.1 Introduction

The package is meant to help scientists to compute the degree of arbuscular mycorrhizal fungi.

All the functions of the package work independently of the colonization scoring system.

0.0.2 Loading the data

To load data in, use the follow command:

readData

0.0.2.1 Example

# Grid
gr <- readData("../inst/extdata/grid.csv") 
# Trouvelot
tr <- readData("../inst/extdata/trouvelot.csv")

0.0.4 Plot data

To summarise the data in a graphical way the package provide 2 plot types:

  • barplot

am_barplot

  • boxplot

am_boxplot

0.0.4.1 Example

p1 <- am_barplot(gr)
p2 <- am_boxplot(gr)
p3 <- am_barplot(tr)
p4 <- am_boxplot(tr)

0.0.5 Save data

To export the summary data, a “save data” function is provided. The function exports both the summary per replicate and the summary per sample in two distinct csv files.

0.0.5.1 Example

am_save(grs, "example_grid.csv")
am_save(trs, "example_trouvelot.csv")
am_save(p1, "barplor_grid.pdf")
## Saving 7 x 5 in image
am_save(p4, "boxplor_trouvelot.pdf")
## Saving 7 x 7 in image
